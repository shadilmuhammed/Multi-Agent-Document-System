# Multi-Agent AI Document System

## Features
- JWT Authentication
- PDF Upload
- AI Chat with Llama 3.2
- ChromaDB Vector Search
- PDF Preview
- Chat History
- Export Chat as PDF
- Source Citations
- User Profile
- Responsive UI

## Tech Stack
Frontend:
- React
- Tailwind CSS
- Axios
- React Router

Backend:
- FastAPI
- SQLAlchemy
- SQLite
- JWT
- ChromaDB
- LangChain
- Ollama (Llama 3.2)
